# CV Website — Florian Beugnies

Site web personnel conçu pour présenter mon curriculum vitae et mes projets à des recruteurs. Il s'agit de mon premier projet personnel réalisé dans le cadre de ma formation en développement web Front End à l'IFAPME.

## Démarche

Le site a été pensé comme un prolongement numérique de mon CV papier : une interface claire, sobre et cohérente visuellement, qui reflète à la fois mon parcours en communication et ma montée en compétences en développement web.

La maquette a d'abord été esquissée sur **Figma**, puis développée de manière itérative — en faisant mes propres recherches, essais et observations au fil du développement.

## Technologies utilisées

- **HTML** — structure des pages
- **CSS** — mise en page, palette de couleurs, dark mode, responsive design
- **JavaScript** — bascule light/dark mode (avec détection des préférences système), effet de frappe animé, galerie lightbox
- **Figma** — maquettage de l'interface
- **Adobe InDesign** — conception du format PDF associé

## Fonctionnalités

- Mise en page deux colonnes (panel fixe + contenu défilant)
- Bascule light / dark mode avec sauvegarde de la préférence utilisateur
- Effet de frappe animé sur le titre
- Navigation par ancres avec scroll-spy
- Pages de projet dédiées
- Design responsive (mobile / tablette)

## Hébergement

Le site est déployé via **GitHub Pages** à partir de ce dépôt.

# CV Website — Florian Beugnies *(English version)*

Personal website designed to present my resume and projects to recruiters. This is my first project built as part of my Front End web development training at IFAPME.

## Approach

The site was conceived as a digital extension of my paper CV: a clear, minimal, and visually cohesive interface that reflects both my background in communication and my growing skills in web development.

The layout was first sketched in **Figma**, then developed iteratively — through my own research, experimentation, and observations alongside my training.

## Technologies used

- **HTML** — page structure
- **CSS** — layout, color palette, dark mode, responsive design
- **JavaScript** — light/dark mode toggle (with system preference detection), typewriter animation, lightbox gallery
- **Figma** — UI wireframing
- **Adobe InDesign** — design of the associated PDF resume

## Features

- Two-column layout (fixed side panel + scrollable content)
- Light / dark mode toggle with user preference saving
- Typewriter animation on the title
- Anchor navigation with scroll-spy
- Dedicated project pages
- Responsive design (mobile / tablet)

## Hosting

The site is deployed via **GitHub Pages** from this repository.

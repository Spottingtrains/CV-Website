# CV Website — Florian Beugnies

Site web personnel conçu pour présenter mon curriculum vitae et mes projets à des recruteurs. Il s'agit de mon premier projet personnel réalisé dans le cadre de ma formation en développement web Front End à l'IFAPME.

## Démarche

Le site a été pensé comme un prolongement numérique de mon CV papier : une interface claire, sobre et cohérente visuellement, qui reflète à la fois mon parcours en communication et ma montée en compétences en développement web.

La maquette a d'abord été esquissée sur **Figma**, puis développée de manière itérative — en faisant mes propres recherches, essais et observations au fil de la formation.

## Technologies utilisées

- **HTML** — structure des pages
- **CSS** — mise en page, palette de couleurs, dark mode, responsive design
- **JavaScript** — bascule light/dark mode (avec détection des préférences système), effet de frappe animé, galerie lightbox
- **Figma** — maquettage de l'interface
- **Adobe InDesign** — conception du CV au format PDF associé

## Fonctionnalités

- Mise en page deux colonnes (panel fixe + contenu défilant)
- Bascule light / dark mode avec sauvegarde de la préférence utilisateur
- Effet de frappe animé sur le titre
- Navigation par ancres avec scroll-spy
- Pages de projet dédiées
- Design responsive (mobile / tablette)

## Hébergement

Le site est déployé via **GitHub Pages** à partir de ce dépôt.
